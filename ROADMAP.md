# Roadmap - Upcoming Features

This document outlines planned features and improvements for Money Flow.

## High Priority

### Multi-User Support
- User authentication and login system
- Multiple user accounts with data isolation
- User profile management
- Password reset functionality

### Mobile Responsive Design
- Optimize UI for mobile devices
- Touch-friendly interactions
- Responsive layouts for tablets and phones
- Mobile-first expense entry

### CSV Import Feature
- Import CSV files to add data to each page/tile
- Support for importing expenses, accounts, assets, investments, plans, and life experiences
- Single CSV file that can populate multiple pages
- Smart CSV parsing that detects and maps columns correctly
- Import validation and error handling
- Preview before importing

### Complete Data Export
- Export entire database as CSV
- Export all data regardless of current page
- Option to export specific categories or all data
- Backup entire financial data in one file
- Export format compatible with import feature

## Medium Priority

### Enhanced Analytics
- Spending trends over time
- Category-wise spending comparisons
- Monthly/yearly reports
- Visual charts and graphs

### Recurring Expenses
- Set up recurring expense templates
- Automatic reminders for fixed expenses
- Recurring expense management UI
- Skip/cancel recurring expenses

### Budget Improvements
- Category-wise budgets
- Budget rollover options
- Budget alerts and notifications
- Budget vs actual comparisons

### Investment Tracking
- Investment performance tracking
- ROI calculations
- Portfolio value tracking
- Investment history graphs

## Low Priority / Future Considerations

### Advanced Features
- Goal-based savings tracking
- Debt tracking and management
- Bill reminders and notifications
- Receipt photo attachments
- Multi-currency support
- Tax reporting features

### Integrations
- Bank account sync (optional, manual-first philosophy maintained)
- Export to accounting software
- Calendar integration for bill reminders

### UI/UX Improvements
- Customizable themes
- Keyboard shortcuts
- Quick expense entry modes
- Dashboard customization

### Technical Improvements
- Performance optimizations
- Offline support (PWA)
- Real-time sync across devices
- API rate limiting
- Comprehensive test coverage

## Contributing Ideas

Have a feature idea? Open an issue on GitHub with the `enhancement` label!

---

**Note**: This roadmap is flexible and may change based on user feedback and community contributions.
